## 1. プロジェクト概要

### 製品コンセプト
- AI駆動の自己実現サポートアプリ
- 大目標を中・小目標に分解し、カンバン形式で管理
- Docker + AWS + Terraform + GitHub Actionsによる本番環境構築

### 主要機能
1. 3画面構成（ログイン/カンバン/マイページ）
2. AIによる目標細分化
3. Markdown連携
4. ドラッグ＆ドロップ機能
