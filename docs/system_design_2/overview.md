# system_design_2/overview.md

## システム概要

### システム名

AI Todo Management

### 目的

AI を活用した Todo 管理システムを提供し、ユーザーの生産性向上を支援する。

### 主な機能

1.  **ユーザー認証:**
    *   ログイン/ログアウト機能
    *   ユーザー登録機能
    *   パスワードリセット機能
    *   二段階認証(将来的に)

2.  **Todo 管理:**
    *   カンバン方式による Todo 管理 (Todo, In Progress, Done の3列)
    *   ドラッグ＆ドロップによるステータス変更
    *   Todo の作成、編集、削除
    *   Todo の検索、フィルター、ソート
    *  アーカイブ機能

3.  **ユーザープロフィール管理:**
    *   プロフィール情報の表示・編集
    *   アバター画像の設定
    *   パスワード変更
    *   テーマ設定(ダークモードとか)
    *   言語設定
    *   ショートカット設定

4.  **AI によるタスク分解支援 (将来実装予定):**
    *   Todo を入力すると、AI が自動的にサブタスクに分解
    *   GPTs連携による高度なタスク分析
    *   AIによる優先度設定の提案
    *  習慣化支援

5.  **管理者機能(将来的には):**
    *   APIドキュメント表示
    *   ログ表示
    *   システム監視
    *   バックアップ管理
    *   IPアクセス制御

### 技術スタック

*   フロントエンド: Next.js (React), TypeScript
*   バックエンド: Node.js, Express
*   データベース: SQLite
*   AI 連携: OpenAI API (将来実装予定)
