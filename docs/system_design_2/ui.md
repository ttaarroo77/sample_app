# 詳細UI設計

本ファイルでは、**画面遷移図**や**画面構成**、**UIワイヤーフレーム**を詳述します。  
概要レベルのコンポーネント紹介は `overview_0/components.md` を参照。

---

## 1. 画面遷移図 (詳細版)

```mermaid
graph TD
    A[ログイン画面] --> B[カンバン画面]
    B --> C[マイページ]
    C --> B[戻る]
    style A fill:#ffe,stroke:#333
    style B fill:#ffe,stroke:#333
    style C fill:#ffe,stroke:#333

ログイン画面: メール & パスワード入力 → 認証成功でカンバンへ
カンバン画面: メインの目標管理UI
マイページ: プロフィール、パスワード変更など

## 2. カンバン画面のUI詳細
ヘッダー: ログアウトボタン or ユーザー情報アイコン
カンバン列: 大目標 / 中目標 / 小目標 など(ドラッグ＆ドロップ)
操作例: 目標追加ボタン、色変更、削除ボタン
オプション:
「AI細分化」ボタン (モーダル)
「MDインポート/エクスポート」ボタン

### 2-1. ワイヤーフレーム例 (簡略)
+-------------------------------------------------+
| Header : [User Icon]  [Logout]                 |
+-------------------------------------------------+
| [ AI Breakdown ] [ MD Import ] [ MD Export ]   |
|------------------------------------------------|
|  [ Big Goals ]    [ Medium Goals ]   [Small?]  |
|   Card1 (color)     Card2 (color)     Card3... |
|   Card4 ...          etc ...                   |
|                   (Drag and Drop)             |
+-------------------------------------------------+


## 3. マイページのUI詳細
フォーム要素:
ユーザー名
パスワード変更 (現行/新規) or Supabase Authに依存
アバター画像URL or ファイルアップロード
+---------------------------------------+
| [ My Page ]                           |
|---------------------------------------|
| Username: [        ]                 |
| Avatar URL: [        ]               |
| Password:  [        ] (Optional)     |
| [ Save Changes ]                     |
+---------------------------------------+


## 4. レスポンシブ対応 (Optional)
スマホ等小画面に対応する場合、カンバンの列を横スクロールor縦レイアウトに切り替えなど。
ポートフォリオのアピールとして導入しても良い。


## 5. その他UI要件
カラーピッカー: 目標色を選択できるUI
モーダル表示: 目標編集 / AI細分化結果 / MDインポート確認など
エラーメッセージ表示: バリデーションエラーなどをToastやAlertで表示
以上が詳細なUI設計の例です。実装時はライブラリ/フレームワークによって微調整してください。
